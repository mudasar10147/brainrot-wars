# Weekly Reports — Brainrot Wars

Weekly detailed reports are **contractually required** under §4.2.

- **Frequency:** One comprehensive report per week.
- **Contract reference:** DocuSign Envelope ID `301F02FD-0BFE-8C76-837F-643FCE87F07A`
- **Failure to submit** may be considered a breach of obligations.

## Required contents (per contract)

Each report must include:

1. **Overall project status**
2. **Team performance**
3. **Completed and pending tasks**
4. **Risk analysis and proposed solutions**

## How to use

1. Copy `Weekly_Report_Template.md` and rename with the report period, e.g.  
   `Weekly_Report_2025-03-10_to_2025-03-16.md`
2. Fill in every section; remove placeholder text.
3. Submit by the agreed weekly deadline to the designated recipient.

## Naming convention (suggested)

- `Weekly_Report_YYYY-MM-DD_to_YYYY-MM-DD.md`  
  Example: `Weekly_Report_2025-03-10_to_2025-03-16.md`
